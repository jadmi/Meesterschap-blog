The history of browsers

Aantekeningen

Een browser bestaat uit meerdere onderdelen: een rendering engine die HTML en CSS parsed en de DOM en AOM opbouwt (zoals Blink/Chromium), een aparte JavaScript engine, een interface en het Accessibility Object Model dat reageert op ARIA.

"Als iets ooit in een browser heeft gewerkt zal het voor eens en altijd werken."

Scripts kun je op drie manieren laden: script voert meteen uit en blokkeert rendering, defer voert later uit en async voert uit wanneer het uitkomt. De browsergeschiedenis gaat van Mosaic (1993) via Netscape, IE, Opera, Safari, Firefox en Chrome naar Safari iOS en het nieuwe Ladybird (2024).
