Hacking things

Aantekeningen

"Hacker" in de positieve zin: alternatieve en creatieve manieren gebruiken om problemen op te lossen. Er is een toenemende afhankelijkheid van Amerikaanse big tech ook voor de overheid. Planned obsolescence werkt zelf repareren en hacken actief tegen.

Low Tech Magazine als inspiratie: dithered images, eco-driven hosting. Static site generators halen content 1x per nacht op, wat minder serverbelasting geeft, leesbaardere code oplevert en markdown als fallback biedt. Je kunt zelf hosten met een oude computer: git repos, messaging platforms, websites.
