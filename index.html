Reflectie (1/3)

Het eerste gesprek vond plaats na Sprint 0, dus de start van de minor was afgerond. Ik vond het zwaar, maar wel leuk en was benieuwd wat me nog meer te wachten ging staan. Ik ben zeer tevreden met mijn inzet tijdens sprint 0 en werkwijze, waarbij ik met zo min mogelijk AI was ik aan de slag gegaan en heb ik nog steeds wat leuks kunnen neerzetten. Het enige waar ik AI voor had gebruikt was wat javascript, maar dat blijf ik moeilijk vinden. Voor de minor had ik al een lange tijd niet meer gecodeerd dus was dit ook een goede opfrisser voor mijn kennis. Na sprint 0 heb ik een goed gevoel over de minor en ben ik benieuwd wat er nog meer gaat komen, de verwachtingen qua inzet en tijdsbesteding zijn hoog, maar ik vind het een leuke uitdaging waarvan ik hopelijk veel kan leren binnen het coderen.

Reflectie (2/3)

Nu we richting het einde van de minor gaan, ben ik blij dat ik de keuze hebt gemaakt om deze minor te volgen. De vakken pakken goed elk een andere interessegebied binnen web development waarbij je binnen het vak de tijd hebt om erin te duiken met de docenten die er veel verstand van hebben en dit graag uitleggen.

Als ik kijk naar de grootste leermomenten tot zo ver is het voor mij vooral de aanpak voor het coderen in het algemeen. Naast specifieke handige stukjes code die ik vaker ga gebruiken zoals clamp is voor mij de werkwijze een stuk efficiënter. Denk aan dingen zoals het document beginnen met de css remedy, een root hebben, variable based te werk gaan en het gebruik van css nesting. Daarnaast gebruik ik nu de basis van emmet wat ook weer tijd bespaard. Ook dingen zoals een overzichtelijke folderstructuur neem ik mee.

Verder vond ik de vakken leuk, HCD was interessant en viel mee qua werk omdat de focus puur ligt op testen en itereren en niet persee de code zelf. BT was straightforward en had ik genoeg de tijd om HTML in te duiken. CSS kostte mij het meeste tijd maar niet persee het lastigst, dat was voor mij API. Ik vind data altijd wel interessant dus vond ik API leuk om mee bezig te ziin, alleen blijft Javascript het lastigst voor mij en kan altijd meer oefening gebruiken. Tot slot ben ik trots op de inzet die ik getoond heb en de uitkomsten als gevolg daarvan. Ik wist wat me te wachten stond met deze echte fulltime minor en die verwachtingen zijn ook zeker waar gemaakt.
