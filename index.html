Reflecties

Reflectie (1/3)

Het eerste gesprek vond plaats na Sprint 0, dus de start van de minor was afgerond. Ik vond het zwaar, maar wel leuk en was benieuwd wat me nog meer te wachten ging staan. Ik ben zeer tevreden met mijn inzet tijdens sprint 0 en werkwijze, waarbij ik met zo min mogelijk AI was ik aan de slag gegaan en heb ik nog steeds wat leuks kunnen neerzetten. Het enige waar ik AI voor had gebruikt was wat javascript, maar dat blijf ik moeilijk vinden. Voor de minor had ik al een lange tijd niet meer gecodeerd dus was dit ook een goede opfrisser voor mijn kennis. Na sprint 0 heb ik een goed gevoel over de minor en ben ik benieuwd wat er nog meer gaat komen, de verwachtingen qua inzet en tijdsbesteding zijn hoog, maar ik vind het een leuke uitdaging waarvan ik hopelijk veel kan leren binnen het coderen.

Reflectie (2/3)

Nu we richting het einde van de minor gaan, ben ik blij dat ik de keuze hebt gemaakt om deze minor te volgen. De vakken pakken goed elk een andere interessegebied binnen web development waarbij je binnen het vak de tijd hebt om erin te duiken met de docenten die er veel verstand van hebben en dit graag uitleggen.

Als ik kijk naar de grootste leermomenten tot zo ver is het voor mij vooral de aanpak voor het coderen in het algemeen. Naast specifieke handige stukjes code die ik vaker ga gebruiken zoals clamp is voor mij de werkwijze een stuk efficiënter. Denk aan dingen zoals het document beginnen met de css remedy, een root hebben, variable based te werk gaan en het gebruik van css nesting. Daarnaast gebruik ik nu de basis van emmet wat ook weer tijd bespaard. Ook dingen zoals een overzichtelijke folderstructuur neem ik mee.

Verder vond ik de vakken leuk, HCD was interessant en viel mee qua werk omdat de focus puur ligt op testen en itereren en niet persee de code zelf. BT was straightforward en had ik genoeg de tijd om HTML in te duiken. CSS kostte mij het meeste tijd maar niet persee het lastigst, dat was voor mij API. Ik vind data altijd wel interessant dus vond ik API leuk om mee bezig te ziin, alleen blijft Javascript het lastigst voor mij en kan altijd meer oefening gebruiken. Tot slot ben ik trots op de inzet die ik getoond heb en de uitkomsten als gevolg daarvan. Ik wist wat me te wachten stond met deze echte fulltime minor en die verwachtingen zijn ook zeker waar gemaakt.

Reflectie (3/3)

// Meesterproef

De meesterproef was een goede afsluiting van deze minor waar alles samenkomt. De samenwerking verliep erg goed en ben ik tevreden over hoe het is gegaan. Het is jammer dat het cijfer wat aan de lage kant was voor het werk wat we erin hebben gestoken, maar ik snap de beredenering erachter. Ik heb gemerkt hoe belangrijk een goede samenwerking is en hoeveel efficiënter je dan te werk kan gaan. Ik heb geleerd mijn sterke punten goed te gebruiken en dit te vertalen naar een sterke oplevering. Qua techniek heb ik goed de kracht van Astro kunnen gebruiken met componenten en props. De toepassing van Javascript voor mijn minigame bleef lastig, maar ik heb het voor elkaar gekregen.

// Leerdoelen

Ik vind dat ik alle drie mijn leerdoelen succesvol heb behaald. Mijn professionele communicatie heb ik ontwikkeld door de rol van contactpersoon op te nemen. De samenwerking verliep professioneel met een duidelijke taakverdeling en gestructureerde communicatie. Voor mijn derde leerdoel heb ik AI gebruikt voor uitleg in plaats van genereren, en ben ik meerdere uren zelfstandig bezig geweest met het oplossen van bugs voordat ik hulp zocht.

// Weekly nerds

De weekly nerds vond ik een leuke toevoeging aan de minor. Binnen de losse nerds vond ik Hans de Zwart het leukst door zijn manier van presenteren en het onderwerp privacy. Op The Web You Want dag 1 was Niels Leenheer mijn favoriet — hij sprak met veel passie over coole projecten. Op dag 2 vond ik Leonie Watson interessant door haar perspectief op AI als hulpmiddel bij een beperking. De bezoeken bij Q42 en De Voorhoede gaven een goed inkijkje in het werkveld.

// Vakken

HCD en CSS vond ik het leukst. Bij HCD was het erg interessant om te interacteren met Ihab en echt te zien hoe hij met het web omgaat. CSS was het zwaarst maar juist ook leuk door de vrijheid die je had. Ik heb mijn creativiteit losgelaten tot in de details en tegelijkertijd veel geleerd.

// What's next?

Alle kennis neem ik mee naar de volgende stappen van mijn front-end loopbaan. Het belangrijkste meeneem punt is component based werken via Astro. Daarnaast kleinere dingen zoals CSS remedy, variable based werken, CSS nesting, emmet en op de juiste manier met AI werken. Als volgende stap wil ik verder met frameworks en meer Javascript oefenen. Ik wil niet stilzitten en deze flow doorzett en ook in mijn vrije tijd aan projecten werken zoals mijn portfolio. Je leert het meeste door te blijven doen!
